# 顏真卿-多寶塔碑
* [字帖連結](https://digitalarchive.npm.gov.tw/Painting/Content?pid=1947&Dept=P)

## 特色
* 用筆多折，橫細直粗特點明顯。
* 橫畫和勾畫的提按明顥，向下頓筆突出。
* 點畫的起筆、頓筆和收筆都保持中鋒行筆的穩健。
* 捺畫的弧度較大，筆毫在行筆時漸漸發力最後儘量鋪開，到停頓處回鋒提筆帶出捺腳。
* 折畫和歐體不同，沒有明顯的「聳肩」現象，折角多呈下垂。
* 和歐體不同，顏體不論筆劃多少，字形大小基本一致。
* 結構緊密嚴謹，平穩端莊，外形飽滿，布白緊湊勻稱。
